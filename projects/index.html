Projects

DC -> SF by bike

In the summer of 2011, a friend and I set out on a two-month bike trip from the outskirts of DC to San Francisco, CA. I saw a lot of America, met incredible people, slept outside a lot, and took a few pictures.

For the trip, I designed and wrote this journal which uses the Google Maps API to pull down elevation data for a given day's route.

FiPy

I worked on FiPy, a numerical analysis platform written in Python, for about a year. This gave me a lot of practice designing and refactoring class hierarchies, debugging segfaulty Python/C errors, and performance tuning. I set up a continuous-integration build system using Buildbot, which was pretty fun.

Soapbox

A while back, GMU's computer science department contracted me to write a tiny CMS that stored announcements and rendered them in various ways throughout the department's website. The platform had to integrate with their existing site (vanilla HTML), had to use flatfile storage, and had to be usable by secretaries.

This was my first real experience designing a UI, and I had good time learning MooTools in order to make an interface that I liked.
